Go 程序的执行由两层组成：Go Program，Runtime，即用户程序和运行时
二者之间通过函数调用来实现内存管理、channel 通信、goroutines 创建等功能
用户程序进行的系统调用都会被 Runtime 拦截，以此来帮助它进行调度以及垃圾回收相关的工作
